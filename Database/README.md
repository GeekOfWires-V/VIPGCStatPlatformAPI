# VIPGC Stat Service

This is the directory that holds the files that can be utilized to build new database services. This is done by configuring the database desired at runtime to support the entire platform.

## Supported Platforms

Database Services supported includes:

- MariaDB
- PostgreSQL
- Microsoft SQL Server
- MongoDB
- Cassandra
- SQLite
- MySQL

## Configuring Web API for Database

This process is not supported in the current state of the API.
